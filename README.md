## Quick Start

### Brain Dump

Say any of these to start:
- `Brain Dump: [topic]`
- `Let's do a Brain Dump on [topic]`

The AI will guide you through a 6-phase process:
1. **Capture** - Topic, goal, templates
2. **Decompose** - Break into learnable chunks
3. **Research** - Find free resources
4. **Document** - Create RAG-ready docs
5. **Integrate** - Load into RAG
6. **Validate** - Gap analysis & sign-off
